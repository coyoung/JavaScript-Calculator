# JavaScript-Calculator

The prompt for this project was to imagine you are a junior software developer working on a new application for a new operating system. A fundamental tool every OS needs is a calculator application. You’ve been tasked with creating a JavaScript program that simulates a calculator capable of performing basic operations and keeping track of all calculations made.

Lab Objectives

Implement functions for addition, subtraction, multiplication, and division.
Store each calculation’s details in an array.
Display the history of calculations to the user.
